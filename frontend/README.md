IntelliType AI is a premium, full-stack web application designed for developers and tech enthusiasts to measure and improve their typing performance. Built with a focus on **Rich UI/UX**, it features cinematic animations, real-time character validation, and deep analytical tracking.

## ‚ú® Key Features

* **Cinematic UI/UX:** High-end "Glassmorphism" interface with animated background textures for an immersive experience.
* **Precision Typing Engine:** 100+ domain-specific sentences focused on AI, ML, and Software Engineering.
* **Real-Time Validation:** Character-by-character highlighting (Emerald Green/Rose Red) with a live accuracy counter.
* **Stopwatch Challenge Mode:** Measures speed (WPM) and accuracy from start to finish, replacing traditional countdowns.
* **Advanced Analytics:** * **Weakest Keys Tracker:** Identifies the top 3 specific keys you consistently miss.
    * **Performance Trend Bar:** A visual chart showing your WPM growth over the last 10 tests.
* **Global Leaderboard:** Compete against other users with a filtered "Top 5" list (max 2 entries per user).
